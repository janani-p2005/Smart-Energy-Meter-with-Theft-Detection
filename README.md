# Smart-Energy-Meter-with-Theft-Detection
📖 Project Description

The Smart Energy Meter with Power Theft Detection is designed to monitor electricity consumption and detect unauthorized usage in real time. This system uses sensors and a microcontroller to measure power consumption and identify abnormal usage patterns. When power theft or unusual consumption is detected, the system alerts the user through buzzer or notification. This helps in reducing electricity wastage and improves efficient energy management.

🎯 Objectives
To monitor electricity consumption in real time
To detect power theft and abnormal usage
To reduce electricity wastage
To provide real-time alerts to users
To improve energy management system

⚙️ Working Principle

The Smart Energy Meter continuously measures voltage and current using sensors. The microcontroller calculates power consumption and compares it with normal usage values. If the system detects abnormal consumption or unauthorized connection, it triggers an alert. The energy usage data is displayed on LCD and can also be sent to user through IoT platform.
